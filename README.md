│

* **tsconfig.json** → TypeScript configuration

## ⚙️ Setup & Run

### 1️⃣ Install dependencies

```
npm install
```

### 2️⃣ Compile TypeScript

```
npx tsc
```

### 3️⃣ Run the compiled code

```
node dist/first.js
```
